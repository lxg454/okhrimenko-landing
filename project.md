# Проект: Личная страница Александра Охрименко

## Что это за проект

Персональная landing page для Александра Охрименко — продюсера онлайн-школ.
Страница представляет автора, указывает его специализацию и предоставляет прямой
способ связи через Telegram. Сайт опубликован на GitHub Pages.

## Для кого

Для потенциальных клиентов и партнёров, которые хотят узнать об Александре
и быстро выйти с ним на связь.

## Страницы

### Уже есть
- `index.html` — главная (и единственная) страница: имя, специализация,
  иконки соцсетей, кнопка «Написать мне»

### Планируются
Дополнительных страниц пока не планируется. Проект остаётся одностраничным.

## Дизайн

- **Тема:** тёмная, фон `#0a0a0a`
- **Акценты:** градиент фиолетовый → голубой → розовый (`#a78bfa`, `#38bdf8`, `#f472b6`)
- **Эффекты:** анимированные фоновые орбы (blur), пульсирующий аватар,
  градиентное свечение имени, hover-анимации на иконках соцсетей,
  пульсирующие кольца вокруг CTA-кнопки
- **Стиль:** минималистичный dark UI, без внешних зависимостей, всё в одном HTML-файле
- **Адаптивность:** корректно отображается на мобильных и десктопе

## Технологии

- Чистый HTML + CSS (без фреймворков и библиотек)
- SVG-иконки (Telegram, Instagram, YouTube) — встроены напрямую в HTML
